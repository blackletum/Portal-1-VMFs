---ОПИСАНИЕ---

Данный архив содержит VMF исходники карт игры Portal 1.
В папке 'Retail' лежат исходники карт релизной версии игры. В папке '2006' лежат тестовые карты разработчиков и старые версии игровых локаций.

---ИНСТРУКЦИИ ПО УСТАНОВКЕ---

1) Извлечь из этого архива папки '2006' и 'Retail' в любое удобное для вас место

2) После распаковки запускаем Hammer, который идет в составе Portal 1 SDK и открываем исходники.


Скачано с сайта https://hl2-beta.ru

 -ENG-

---DESCRIPTION---

This archive contains VMF source files of the Portal 1 maps. 
The Retail folder contains the sources of the retail version's maps, and the 2006 folder contains source files of the devtest maps, and the older test chamber maps created by the developers.


---INSTALLATION INSTRUCTIONS---

1) Unpack the '2006' and 'Retail' folders from this archive

2) After unpacking this archive, run the Portal 1 SDK's Hammer and open the .vmf files


Downloaded from https://hl2-beta.ru
